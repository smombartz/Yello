# Welcome to Ello

Ello is a personal contact manager that imports contacts from VCF files and provides powerful tools for organizing, deduplicating, and managing large contact databases. Built for single users managing 10,000+ contacts with photos.

## Why Ello?

Existing contact managers either lock you into an ecosystem (Apple, Google) or are bloated with features you don't need. Ello is different:

## Key Features

### Contact Import
Import contacts from VCF files (vCard 2.1, 3.0, and 4.0). Supports single or multi-contact files up to 100MB with automatic photo extraction and processing.

### Fast Search
Full-text search across names, emails, companies, and phone numbers. Results appear as you type with prefix matching support.

### Deduplication
Five detection modes to find duplicates:
- **Email** - Exact email address matches
- **Phone** - Normalized phone number matches (E.164)
- **Address** - Street address matches
- **Social** - Social profile matches
- **Recommended** - Multi-signal AI matching with confidence scores

### Contact Cleanup
Identify contacts that need attention:
- Truly empty contacts (no data at all)
- Name-only contacts (name but no contact info)
- Problematic contacts (suspicious patterns like many domains)

### Archive & Restore
Soft-delete contacts without losing them permanently. Archive, restore, or export archived contacts as VCF.

### Groups & Categories
Organize contacts by category. Filter and view contacts by group.
