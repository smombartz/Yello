# Contacts API

Endpoints for managing contacts.

## List Contacts

Retrieve a paginated list of contacts with optional search and filtering.

<ParamField query="page" type="number" default="1">
  Page number (1-indexed)
</ParamField>

<ParamField query="limit" type="number" default="50">
  Contacts per page (max 100)
</ParamField>

<ParamField query="search" type="string">
  Full-text search query across name, email, company
</ParamField>

<ParamField query="category" type="string">
  Filter by category name
</ParamField>

<RequestExample>
```bash cURL
curl "http://localhost:3000/api/contacts?page=1&limit=50&search=john"
```

```javascript JavaScript
const response = await fetch('/api/contacts?page=1&limit=50&search=john');
const data = await response.json();
```
</RequestExample>

<ResponseExample>
```json Response
{
  "contacts": [
    {
      "id": 1,
      "displayName": "John Smith",
      "company": "Acme Inc",
      "primaryEmail": "john@acme.com",
      "primaryPhone": "(555) 123-4567",
      "photoUrl": "/photos/thumbnail/a1/a1b2c3d4.jpg"
    }
  ],
  "total": 1523,
  "page": 1,
  "totalPages": 31
}
```
</ResponseExample>

## Update Contact

Update a contact's basic information.

<ParamField path="id" type="number" required>
  Contact ID
</ParamField>

<ParamField body="displayName" type="string">
  Display name
</ParamField>

<ParamField body="company" type="string">
  Company name
</ParamField>

<ParamField body="title" type="string">
  Job title
</ParamField>

<ParamField body="notes" type="string">
  Notes
</ParamField>

<ParamField body="birthday" type="string">
  Birthday (YYYY-MM-DD format)
</ParamField>

<RequestExample>
```bash cURL
curl -X PUT "http://localhost:3000/api/contacts/123" \
  -H "Content-Type: application/json" \
  -d '{
    "displayName": "John D. Smith",
    "company": "Acme Corporation",
    "title": "Chief Executive Officer"
  }'
```
</RequestExample>

<ResponseExample>
```json Response
{
  "id": 123,
  "displayName": "John D. Smith",
  "company": "Acme Corporation",
  "title": "Chief Executive Officer",
  "notes": "Met at conference 2024",
  "updatedAt": "2025-01-21T14:22:00Z"
}
```
</ResponseExample>

## Response Schemas

### Contact List Item

| Field | Type | Description |
|-------|------|-------------|
| `id` | number | Contact ID |
| `displayName` | string | Formatted display name |
| `company` | string | Company name |
| `primaryEmail` | string | Primary email address |
| `primaryPhone` | string | Primary phone (formatted) |
| `photoUrl` | string | Thumbnail photo URL |

### Contact Detail

| Field | Type | Description |
|-------|------|-------------|
| `id` | number | Contact ID |
| `firstName` | string | First name |
| `lastName` | string | Last name |
| `displayName` | string | Display name |
| `company` | string | Company |
| `title` | string | Job title |
| `notes` | string | Notes |
| `birthday` | string | Birthday (YYYY-MM-DD) |
| `emails` | array | Email addresses |
| `phones` | array | Phone numbers |
| `addresses` | array | Street addresses |
| `socialProfiles` | array | Social media profiles |
| `categories` | array | Category memberships |
| `instantMessages` | array | IM handles |
| `urls` | array | Website URLs |
| `relatedPeople` | array | Related contacts |
| `photoUrl` | string | Photo URL |
| `createdAt` | string | Creation timestamp |
| `updatedAt` | string | Last update timestamp |
